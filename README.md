# 🍽️ Project Restaurant

A modern restaurant website created with React, JavaScript and CSS.  
The project presents a stylish landing page for a restaurant with menu sections, navigation, and interactive UI elements.

## 🌐 Live Demo

🔗 https://solichka27.github.io/project_restaurant/

---

## 📌 Features

- Modern restaurant UI
- Navigation menu
- Stylish food sections
- Smooth layout and clean interface
- Interactive elements with React

---

## 🛠️ Technologies Used

- React
- CSS
- JavaScript

## 🚀 Getting Started

To run this project locally:

### 1. Clone repository

```bash
git clone https://github.com/Solichka27/project_restaurant.git
```

### 2. Open project folder

```bash
cd project_restaurant
```

### 3. Run project

Write in the console `npm run start`.
